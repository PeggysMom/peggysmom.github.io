automaton robo-composer


Simple generative synth with fucked up UI.
guide: ***SAY NO TO ALL PERMISSIONS EXCEPT SOUND CARD USE***
To begin, you need to toggle "start DSP" and set a speed between 0 and 2 (slower/under 1 is recommended) then you must select a note on the RED keyboard and either toggle "Auto-compose" or select a note on the YELLOW keyboard. Thats it, have fun. Also you need to click on most of the UI objects for them to start funcionting properly.
